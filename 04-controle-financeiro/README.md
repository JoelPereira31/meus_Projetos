# 💰 Controle Financeiro Pessoal (Projeto 04)

Este é o quarto projeto do meu portfólio de desenvolvimento. Trata-se de uma aplicação web dinâmica para gerenciamento de finanças pessoais. O sistema permite ao usuário adicionar receitas e despesas, calculando o saldo geral em tempo real.

## 🚀 Funcionalidades

*   **Adição de Transações:** Inserção de receitas (valores positivos) e despesas (valores negativos) com validação de dados.
*   **Cálculo Dinâmico:** Atualização instantânea do saldo atual, total de ganhos e total de gastos usando métodos modernos de array do JavaScript.
*   **Histórico Interativo:** Listagem das transações com a opção de exclusão individual.
*   **Persistência de Dados:** Uso do `LocalStorage` do navegador para garantir que os dados não sejam perdidos ao recarregar ou fechar a página.

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica e formulários.
*   **CSS3:** Estilização moderna, variáveis CSS, transições e layout responsivo.
*   **JavaScript (ES6+):** 
    *   Manipulação do DOM.
    *   Armazenamento em banco de dados local (`LocalStorage`).
    *   Manipulação avançada de arrays (`map`, `filter`, `reduce`).

## 💡 Aprendizados

Durante este projeto, pude consolidar o entendimento sobre como tratar entradas de usuários, evitar comportamentos indesejados de formulários e utilizar a lógica matemática do JavaScript para separar e acumular valores com precisão.
