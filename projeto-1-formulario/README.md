# 01 - Formulário de Cadastro Inteligente 📝

Este é o primeiro projeto da minha trilha de evolução em Desenvolvimento Web e Engenharia de Software. Trata-se de um formulário de cadastro moderno com validação de dados em tempo real e análise de força de senha.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica do formulário.
- **CSS3**: Estilização moderna com variáveis, flexbox, transições de foco e modo escuro (Dark Mode).
- **JavaScript (ES6+)**: Validação dinâmica de campos via manipulação do DOM e expressões regulares (RegEx).
- **Font Awesome**: Ícones vetoriais de status.

## ⚙️ Funcionalidades implementadas
- [x] Validação de nome em tempo real (mínimo de 3 caracteres).
- [x] Validação de formato de e-mail por expressão regular.
- [x] Medidor dinâmico de força de senha baseado em 4 critérios (tamanho, maiúsculas, números e caracteres especiais).
- [x] Validação de compatibilidade de senhas (confirmação).
- [x] Desbloqueio inteligente do botão de envio apenas quando todos os critérios forem válidos.
- [x] Simulação de envio com reset visual dos estados.

## 🛠️ Como rodar o projeto
1. Clone este repositório.
2. Abra a pasta `01-formulario-inteligente` no seu navegador através do arquivo `index.html` ou utilizando uma extensão de servidor local (como Live Server).
